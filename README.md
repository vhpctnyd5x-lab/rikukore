# 陸これ（仮） — 陸版艦これ

陸上自衛隊の装備を擬人化した育成＆戦闘ブラウザゲーム。参考文献：**りっくじあーす**。

## 遊び方
通常は `起動.command` をダブルクリックして起動します。VOICEVOXが入っていれば音声も自動で準備します。データは `localStorage` に保存されます。

> ローカルで `fetch` が制限される場合は、このフォルダで簡易サーバを起動：
> ```
> cd RikuKanColle && python3 -m http.server 8765
> # → http://127.0.0.1:8765/src/index.html
> ```

## 画面
- **駐屯地**: 状況確認・補給・リセット
- **編成**: 保有隊員を小隊（最大4名）に編入
- **配備**: 資源を投入して新隊員を獲得（投入量でレア度↑）
- **出撃**: 演習場で戦闘、勝利で資源＆経験値
